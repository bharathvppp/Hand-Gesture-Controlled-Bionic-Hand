# Hand-Gesture-Controlled-Bionic-Hand
This project demonstrates a vision-based bionic hand that mirrors human finger movements in real time using MediaPipe hand tracking and Arduino-controlled servo motors. The system detects hand landmarks through a webcam, interprets finger positions, transmits movement data over serial communication, and actuates an InMoov 3D-printed robotic hand using MG996R servo motors.
